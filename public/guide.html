Setup
Go Live in 4 Steps
The system runs in Demo Mode out of the box. Follow these steps to connect real services.
1
Connect your database (Supabase)
Create a free project at supabase.com, then copy the connection string into your Vercel environment variables as DATABASE_URL. Run the schema migrations in schema.sql to create all tables automatically.
2
Add your Twilio credentials for SMS
Sign up at twilio.com, get a phone number, then set TWILIO_SID, TWILIO_TOKEN, and TWILIO_FROM in Vercel. SMS will activate automatically — no code changes needed.
3
Set your Google Business review link
Go to your Google Business Profile → Get more reviews → copy the direct link. Set it as GOOGLE_REVIEW_URL in Vercel. Clients who rate 4–5★ will see a button that opens directly to your Google review form.
4
Embed the reviews widget on your website
Paste two lines of code anywhere on your Webflow or website page. The widget auto-loads verified 4–5★ reviews with client names and comments. Supports light and dark themes.

Website Integration
Embedding Reviews on Your Website
Add the following two lines anywhere in your page HTML. Works in Webflow custom code, WordPress, or any site builder.
<!-- 1. Target container (place where you want the widget) -->
<div id="arise-reviews"></div>

<!-- 2. Widget script (paste before </body>) -->
<script
  src="https://YOUR-DOMAIN.com/widget.js"
  data-limit="6"         <!-- number of reviews to show -->
  data-theme="light"     <!-- "light" or "dark" -->
  data-target="arise-reviews" <!-- matches the div id above -->
></script>

FAQ
Common Questions
Does this work without real API keys?
Yes — the system runs in Demo Mode automatically when no DATABASE_URL is set. All features are fully testable with in-memory mock data. Click "Load Sample Data" on the dashboard to populate it.
Can clients opt out of feedback messages?
Yes. Every feedback request includes an unsubscribe link. Clicking it marks the client as opted-out and they won't receive future messages. You can also toggle opt-out manually per client in the Clients page.
What happens to negative reviews — do they go to Google?
Never. Ratings of 1–3★ are routed internally only. The client sees a warm message saying the team will follow up. Only 4–5★ clients are prompted to leave a Google review.
How are feedback requests triggered?
When a visit is recorded via the Clients page (or via API), a feedback request is created and sent automatically. Cron jobs run daily to process pending requests and send follow-up reminders.
Are client names shown publicly in the widget?
No — the widget shows only the first name and last initial (e.g. "Margaret J."), never the full name. Comments are shown verbatim only if they were submitted.
Can I customize the message templates?
Yes — go to Settings → Messaging to edit the SMS and email templates. Templates support {client_name}, {caregiver_name}, and {link} variables.
